## App Store description

VerifyAI is positioned as an online-safety image search for dating, remote networking, suspicious sellers, and other profiles. The listing highlights:

- Deep multi-engine search across Google, Yandex, Google Lens, and Shutterstock.
- Catfish and scam checks based on reused photos, known reports, and stolen identities.
- Stock-photo recognition.
- iOS screenshot/share workflow for images from dating, messaging, and marketplace apps.
- Evidence transparency: source links and a visible analysis pipeline.
- Exact-image, perceptual-hash, crop, and face-comparison signals.
- AI-generated image detection presented as reference information, not certainty.
